近期读书总结：Python/MySQL/HTML/CSS/XML

发表于 2017/06/10 | 分类于 Reading |

前言

第一次写读书总结。其实就是突然有个念头，想把每次看过的一些书记录一下，做个总结，以后也还会继续。

现在，我转换了读书的方式，有的书，适合精读，需要慢慢品味，而有的书，没那么多时间来精读，或者说白了，你没那么多时间来系统的进行学习。比如对于计算机方面的一些书，如果每一门技术都要花上很长时间进行系统的学习，那么你能学的东西很少很少，所以最好的学习方法就是很快的浏览，快速入门，然后进行练习，在练习的过程中继续学习之前不懂或没注意到的东西。

最近其实读了两本书，或者说三本吧，做一个简单的总结。

《Python 基础教程》

《Python 基础教程》

  • 作者：Magnus Lie Hetland
  • 出版社：人民邮电出版社

本书也是一本 Python 的入门书，不过讲的是 Python 2 。

这本书是在去年暑假的时候买的，之后零零散散看过一些，直到这个学期才心血来潮，基本看完了一遍。

首先，不得不说，Python 真的很强大，开发效率很高，有很多现成的库可用，当然，它也不是万能的，比如没人能够用它来写操作系统。

无论 Python 还是别的东西，都是入门简单，精通难，不过最近总算用 Python 写了一些可以算得上有用的东西吧，也学习了如何用 Python 来操作 MySQL 数据库 。

《MySQL 入门很简单》

《MySQL 入门很简单》

  • 作者：黄缙华
  • 出版社：清华大学出版社

正如书名所表达的意思，这是一本 MySQL 的入门书，全书讲的东西比较简单，例子很直观，可以带领一个从未接触过数据库的人快速走进 MySQL 的世界。

通过阅读本书，能够掌握 MySQL 数据库的基本操作、基本命令，也能够编写一些简单的 SQL 脚本。当然，这是一本入门书，所以，如果想要深入学习 MySQL 的话仅靠这本书可能不够。

《Head First HTML 与 CSS》

《Head First HTML 与 CSS》

  • 作者：罗布森 (Elisabeth Robson) 弗里曼 (Eric Freeman)
  • 出版社：中国电力出版社

首先，不得不说，这本书最大的缺点就是：废话太多，废话太多，废话太多。

重要的事说三遍。

当然，里面讲到的关于 HTML 和 CSS 的东西应该还是比较全面的，只是可能这一系列的书就是想让学习变得有趣，于是加了很多对我而言可能是废话的东西进去。所以导致书真沉，700 多页的东西，如果把废话去除了，可能不到 300 页。

之所以想学 HTML ，是觉得毕竟作为一名计算机系的学生，就算以后不搞前端，好歹也要能写一些简单的网页，要看的懂简单的网页源代码。当然，内心还有一个比较大胆的想法，就是之后打算写一个 Hexo 博客的主题，不过这是后话。
